# Vaultria

Vaultria is a free language learning web app focused on Japanese, Korean, and Spanish.
The goal is to create a simple platform where anyone can learn these languages without needing expensive subscriptions.

## Current Features

* Lessons for Japanese, Korean, and Spanish
* Practice exercises
* Progress tracking
* Text-to-speech for pronunciation

## Project Status

Vaultria is currently in early development and may contain bugs.
I'm actively improving it and adding new lessons and features.
